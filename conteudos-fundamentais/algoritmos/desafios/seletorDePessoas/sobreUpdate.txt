Este programa é um sistema simples para registrar informações sobre pessoas, permitindo ao usuário inserir detalhes como sexo, idade e cor do cabelo. Depois que o usuário inserir as informações de várias pessoas, o programa exibirá quantos homens têm mais de 18 anos com cabelo castanho e quantas mulheres têm entre 25 e 30 anos com cabelos loiros.

Validação de Entrada: 
Adicionar validação para garantir que as entradas do usuário estejam dentro de intervalos válidos, por exemplo, garantir que a idade esteja dentro de um intervalo razoável e que a opção selecionada para cor do cabelo seja válida.

Opção de Alterar Dados: 
Permitir que o usuário altere os dados de uma pessoa após inseri-los, caso cometa algum erro.

Exibir Informações Detalhadas: 
Além de exibir os resultados finais, você pode adicionar a funcionalidade de exibir informações detalhadas sobre as pessoas registradas, como listar todas as pessoas cadastradas com seus respectivos detalhes.

Utilizar Enumerações para Cor do Cabelo: 
Em vez de usar números para representar a cor do cabelo, você pode usar uma enumeração ou um conjunto de constantes para representar as opções disponíveis. Isso tornaria o código mais legível e menos propenso a erros.

Interface Gráfica: 
Criar uma interface gráfica simples (GUI) em vez de usar prompts, o que pode tornar a interação do usuário mais amigável.

Armazenamento de Dados: 
Implementar uma funcionalidade para armazenar os dados das pessoas em um banco de dados ou em armazenamento local (como localStorage ou sessionStorage) para que os dados persistam entre as execuções do programa.

Adicionar Mais Estatísticas: 
Além das estatísticas atuais, você pode adicionar mais métricas e análises, como a média de idade das pessoas registradas ou a distribuição das cores de cabelo.

Refatoração do Código: 
Avaliar o código atual e identificar oportunidades para refatoração, como a separação de responsabilidades em funções mais específicas e a redução de duplicação de código.
